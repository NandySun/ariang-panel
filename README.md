# 下载 · HanaAgent 插件

aria2 下载管理面板，HanaAgent 原生插件。

## 功能

- 实时查看下载任务：活动 / 等待中 / 已停止 三区分类
- 进度条、速度、剩余时间、连接数、做种数
- 单任务暂停 / 继续 / 删除
- 批量操作：全部暂停、全部继续、清除已完成
- 快捷添加：右下角 "+" 按钮粘贴 URL 或磁力链接

## 技术要点

- 零外部依赖，纯 HTML/CSS/JS
- 前端通过同源代理路由与 aria2 RPC 通信，无 CORS 问题
- 跟随 HanaAgent 主题自动切换明暗色
- 轮询 2 秒刷新，时间 1 秒独立更新

## 安装

1. 下载 `ariang-panel.zip`
2. HanaAgent → 设置 → 插件 → 从本地安装
3. 选择 zip 文件安装
4. 侧边栏出现「下载」图标

## 前置条件

- 系统已安装 aria2 并开启 RPC（默认端口 6800）
- 如需密钥，在插件设置中配置 RPC 密钥

## 配置

| 选项 | 默认值 | 说明 |
|------|--------|------|
| Aria2 RPC 地址 | `http://localhost:6800/jsonrpc` | aria2 JSON-RPC 端点 |
| RPC 密钥 | 空 | aria2 的 rpc-secret |
