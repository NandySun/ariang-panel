# AriaNg 插件重建简报（最终版）

## 当前状态

v1.0.0 已完成，功能完整可用。

## 已实现功能

**核心面板**
- 任务列表：活动 / 等待中 / 已停止 三区分类展示
- 进度条 + 百分比 + 速度 + 剩余时间
- 单任务操作：暂停 / 继续 / 删除（二次确认）
- 卡片展开详情：连接数 / 做种数 / 上传速度
- 全局统计：下载速度、上传速度、任务计数

**快捷操作**
- 右下角 "+" 浮按钮 → 弹出输入框 → 粘贴 URL 或磁力链接添加下载
- 批量操作栏：全部暂停 / 全部继续 / 清除已完成（二次确认）

**交互设计**
- 点击卡片任意位置展开/收起详情
- 点击页面空白处收起全部
- 展开状态跨轮询保持（不会闪烁或自动缩回）
- 右上角时间每秒独立更新

**技术架构**
- 前端 fetch 走同源 `/rpc` 代理路由，绕过 CORS 跨域限制
- 代理路由自动注入 RPC 密钥（从插件配置读取）
- 页面认证凭证从 URL query 透传到代理请求

## 关键踩坑

1. **CORS 跨域**：页面在 Hana iframe 内 fetch `localhost:6800` 被浏览器拦截，加同源代理路由解决
2. **代理路由需认证**：fetch 没带 URL 凭据返回 403，将页面 query string 注入 `__AR2_URL__` 解决
3. **轮询重渲染导致 UI 闪动**：移除 hover 动画的 transition（border-color / transform / opacity），用 `_expanded` 字典跨帧保持展开状态
4. **dev 插件覆盖社区版**：侧边栏开关失效，因为 dev 版覆盖了社区版的路由
